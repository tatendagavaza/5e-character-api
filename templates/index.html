DnD 5e character API

A RESTful API to generate random 5th edition Dungeons and Dragons characters of 1st level. The source code is on Github

Generate Random 1st level character

Send a GET request to:

/random-character

Search for specific class

Send a GET request to:

/get-class?class="{class}"

Search for specific background

Send a GET request to:

/get-background?bg="{background}"
